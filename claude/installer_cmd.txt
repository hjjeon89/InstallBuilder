3. D:\__work\__src\gitea\ModuleCycler_2025\SubApps\CyclerDataAnalyzer\CyclerDataAnalyzer\CyclerDataAnalyzer.csproj
4. D:\__work\__src\gitea\CANDBEditor\CANDBEditor\CANDBEditor.csproj

install 파일은 아래 경로에 생성해줘.
1. D:\__work\__src\gitea\ModuleCycler_2025\install
zip 파일이 아닌 실제로 프로그램을 설치하는 exe 파일을 만든다.

[추가 요구사항]
1. exe파일로 설치가 진행되면 GreenPowerCycler.exe가 바탕화면에 바로가기로 생성되도록 한다.
2. exe파일로 설치가 진행되면 ScheduleEditor.exe가 바탕화면에 바로가기로 생성되도록 한다.
3. exe파일로 설치가 진행되면 CyclerDataAnalyzer.exe가 바탕화면에 바로가기로 생성되도록 한다.
4. exe파일로 설치가 진행되면 CANDBEditor.exe가 바탕화면에 바로가기로 생성되도록 한다.

모든 작업이 완료되면 정상적으로 실행되는지 확인한다.

Version 1.0까지는 완료됨.======================================================================

[요구사항]
1. 사용자가 등록하는 dll 파일들을 정해진 위치로 복사하는 기능 추가
1-1. 여러 개의 dll 파일을 등록할 수 있어야한다.
1-2. Drag & Drop으로 등록할 수 있어야 한다.
1-3. 파일 탐색기로 등록할 수 있어야 한다.
1-4. 프로그램 설치 시 dll 파일들이 복사될 위치를 설정할 수 있어야 한다.
2. 사용자가 등록하는 파일들을 정해진 위치로 복사하는 기능 추가(config, text, json 파일들)
2-1. 프로그램 설치 시 등록한 파일들이 복사될 위치를 설정할 수 있어야 한다.
2-2. Drag & Drop으로 등록할 수 있어야 한다.
2-3. 파일 탐색기로 등록할 수 있어야 한다.

[추가 요구사항-001]
1. Dll 파일을 등록한다고 해서 확장자가 dll일 필요는 없다. dll과 같이 등록되어야 하는 파일들도 존재한다. 필터 기능을 삭제한다.
2. 사용자가 설치될 파일의 버전을 직접 입력할 수 있어야 한다.
2-1. 버전은 1.0.0.0을 기준으로 한다.
3. 여러 프로젝트를 한꺼번에 설치파일은 만들진 않는다. 한개만 등록할 수 있어도 충분하다.
